# 🔐 Responsive Login Page

A modern and responsive Login Page built using **HTML5** and **CSS3**. This project features a clean, professional user interface with a responsive layout that works smoothly across desktops, tablets, and mobile devices.

## ✨ Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive layout
- 🔒 Professional login form
- 🖱️ Smooth hover effects
- ⚡ Fast loading and lightweight
- 💻 Cross-browser compatible
- 📐 Built with semantic HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
